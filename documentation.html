üé® Customization Guide

Changing Colors

To change the primary color scheme:

  1. Open css/style.css in a text editor
  2. Find the :root section at the top
  3. Update the CSS variables: 
    :root {
    --primary-color: #667eea;    /* Change this */
    --secondary-color: #764ba2;  /* Change this */
    --text-color: #333333;
    --bg-color: #ffffff;
}
  4. Save the file and refresh your browser

Changing Logo

  1. Prepare your logo image (recommended: PNG with transparent background)
  2. Name it logo.png
  3. Upload to the images/ folder
  4. Open index.html (and all other HTML files)
  5. Find the logo image tag: 
    <img src="images/logo.png" alt="Company Logo">
  6. Update the src attribute if your logo has a different name

Changing Text Content

  1. Open any HTML file in a text editor
  2. Find the text you want to change
  3. Replace it with your own content
  4. Save and upload to your server
Tip: Use Find & Replace (Ctrl+H) in your text editor to change text across multiple files quickly.

Changing Images

  1. Prepare your images (recommended: WebP or JPG format)
  2. Optimize images for web (use tools like TinyPNG.com)
  3. Upload to images/ folder
  4. Update image references in HTML: 
    <img src="images/your-image.jpg" alt="Description">

Changing Fonts

  1. Choose a Google Font from fonts.google.com
  2. Add the font link to the <head> section: 
    <link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">
  3. Update css/style.css: 
    body {
    font-family: 'Roboto', sans-serif;
}

üìß Contact Form Setup

Important: The contact form requires server-side processing to send emails. You'll need to configure this based on your hosting environment.

Option 1: PHP Mail (Most Common)

  1. Create a file named contact-handler.php
  2. Add this code (update email address): 
    <?php
if($_POST) {
    $name = $_POST['name'];
    $email = $_POST['email'];
    $message = $_POST['message'];
    
    $to = "your-email@example.com";  // Change this
    $subject = "Contact Form Submission";
    $body = "Name: $name\nEmail: $email\n\nMessage:\n$message";
    
    if(mail($to, $subject, $body)) {
        echo json_encode(['success' => true]);
    } else {
        echo json_encode(['success' => false]);
    }
}
?>
  3. Upload to your server
  4. Update contact.html form action: 
    <form action="contact-handler.php" method="POST">

Option 2: Third-Party Services

Use services like Formspree, EmailJS, or SendGrid:

  1. Sign up for a free account
  2. Get your form endpoint URL
  3. Update the form action attribute
  4. Follow service-specific instructions

Form Validation

The template includes built-in JavaScript validation. To customize:

  1. Open js/main.js
  2. Find the validateForm() function
  3. Modify validation rules as needed

Common Issues & Solutions

Issue: Images not showing

Solution: Check that image paths are correct and files were uploaded properly. Use relative paths like images/photo.jpg.

Issue: Contact form not working

Solution: Ensure you've configured the PHP handler correctly and your hosting supports PHP mail function.

Issue: Mobile menu not opening

Solution: Check that js/main.js is loaded properly and there are no console errors.

Issue: Layout breaks on mobile

Solution: Ensure css/responsive.css is linked and viewport meta tag is present in HTML head.
